## 이론적 근거 작성 규칙 (최우선)
- 반드시 [참고 교재 내용]에서 근거를 찾아 인용한다
- 형식: 이론적 설명 1~2문장 + (저자, 연도) [교재명 p.페이지]
  예) 체액 부족 시 빈맥이 먼저 나타난다.
      (Canfield, 2021) [간호진단과 근거기반 간호중재 p.120]
- 참고 교재 내용에서 찾지 못한 중재는 작성하지 않고
  출처가 확인된 다른 중재로 대체한다
- Claude 자신의 기억만으로 출처를 만들어내지 않는다

## 간호진단 규칙
- 형식: [원인구]와 관련된 [문제구]
- 의학적 진단명을 원인구로 사용 절대 금지
  틀림: "충수염과 관련된 급성통증" ❌
  옳음: "생물학적 손상 요인과 관련된 급성통증" ✅
- NANDA-I 분류체계 기반으로 진단
- 2~3개 수립, 아래 기준으로 우선순위 결정:
  1) 생명·안위 위협 여부
  2) 실제 문제 > 잠재적(위험) 문제
  3) Maslow 욕구위계
     1순위: 생리적 욕구 (체액, 호흡, 영양, 순환, 체온)
     2순위: 안전·안정 욕구 (통증, 감염, 낙상)
     3순위: 사랑·소속 욕구
     4순위: 자존감 욕구
     5순위: 자아실현 욕구
  4) 대상자가 가장 중요하다고 인식하는 문제 고려
- 각 진단마다 해당 O/S 근거 자료 선별 제시
- 영역/과 번호 반드시 표기
  예) 영역 12. 안위 / 과 1. 신체적 안위

## 간호목표 규칙
- 장기목표 1개 (퇴원 시까지)
- 단기목표 3개 (48시간 이내, 3일 이내, 5일 이내)
- 형식: 대상자 + 목표시간 + 수행기준 + 측정가능한 동사
- 주어는 반드시 "대상자" (간호사 X)
- 수행기준은 반드시 수치 또는 관찰 가능한 행동
  예) 대상자는 48시간 이내에 NRS 3점 이상 감소되었다고 말한다.
  예) 대상자는 퇴원 시까지 NRS 2점 이하를 유지한다.

## 간호계획 규칙 (총 9개 필수)
반드시 아래 3가지로 나눠서 각 3개씩 작성:

[진단적 계획 3개]
- 환자 상태 관찰·평가 활동
- 허용 동사만 사용:
  사정한다, 관찰한다, 측정한다,
  모니터링한다, 확인한다, 조사한다

[치료적 계획 3개]
- 문제 해결을 위한 직접 간호 중재
- 허용 동사만 사용:
  수행한다, 조절한다, 유지한다,
  투여한다, 실시한다

[교육적 계획 3개]
- 환자·보호자 교육 중재
- 허용 동사만 사용:
  설명한다, 교육한다, 시범 보인다,
  격려한다, 안내한다

각 계획마다 이론적 근거 필수 작성
형식: 이론적 설명 + (저자, 연도) [교재명 p.N]

## 간호수행
간호계획 동사를 과거형으로 변환하여 동일하게 작성
사정한다 → 사정하였다
관찰한다 → 관찰하였다
측정한다 → 측정하였다
모니터링한다 → 모니터링하였다
확인한다 → 확인하였다
수행한다 → 수행하였다
유지한다 → 유지하였다
투여한다 → 투여하였다
실시한다 → 실시하였다
설명한다 → 설명하였다
교육한다 → 교육하였다
시범 보인다 → 시범 보였다
안내한다 → 안내하였다
격려한다 → 격려하였다

## 간호평가
각 목표(장기 1개 + 단기 3개)의 달성 여부 서술
평가 결과: 목표 달성 / 부분 달성 / 미달성
미달성 시 간호계획 재수립 필요 명시
